1. 创建Vue工程


    # 安装vue cli
    npm install -g vue-cli
    
    # Initialize a webpack project in the directory 'yz_agent'
    vue init webpack  yz_agent
    
    # Change into the directory
    cd yz_agent
    
    # Install dependencies
    npm install
    
    # Fire up the dev server with HMR
    npm run dev
